The gole of this Application ...

This application is made to practice the basics of the Angular Framework, and it is a part of the PSEU training.

In this application, I applied the following concepts/ideas.
  • Routing between components.
  • How to deal with service and inject it into components using the Dependency injection.
  • How to contact with a Firebase.
  • Applying the CRUD operation with Firestore cloud on firebase.
  • How to write a good form with validations on the fields.
